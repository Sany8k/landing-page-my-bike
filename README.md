# MyBike landing page

MyBike is a fully responsive, modern landing page designed to effectively showcase a product.

## Live Demo

Experience the live website: [MyBike](https://sany8k.github.io/landing-page-my-bike/)

## Design Reference

[Design Reference](https://www.figma.com/design/NZQAIydtHo5QkINyGLHNcq/BIKE-New-Version?node-id=0-1&p=f&t=UW9CzXWEXc4plCRJ-0)

## Technologies Used

**Core**
* HTML5 - markup
* Sass - Styling

**Development & Deployment**
* Parcel (v5.3.1) - Build tool
* ESLint (v8.57.0) - Code quality
* Github Pages (v6.1.1) - Hosting and deployment

## Getting Started

1. Clone the repository:
   
```bash
git clone https://github.com/Sany8k/landing-page-my-bike.git
cd landing-page-my-bike
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the project locally:

```bash
npm start
# or
yarn start
```
